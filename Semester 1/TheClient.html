Wat heb ik vandaag gedaan?

Ik ben vandaag begonnen aan de CSS Layout Modes opdracht. https://github.com/fdnd-task/layout-in-css/blob/main/docs/INSTRUCTIONS.md.

Voor deze opdracht moeten we de layouts één voor één nabouwen. Je heb 8 'basic' layouts en 8 'advanced' layouts. Ik ben vandaag begonnen met 'Basic-layout 1'.

Ik ben inmiddels al een beetje bekend met de gewone grid (grootte veranderen, border radios, responsive maken, enz.), Omdat ik hieraan werkte tijdens de eerste sprint. maar ik moet nog meer leren om de 18 layouts in deze opdracht goed na te kunnen bouwen. Ik ben daarom video tutorials gaan bekijken over 'flexbox'.
